# Mentoring 09.01.2026

## OOP (Object Oriented Programming)

- [Klassen als "Baupläne" für Objekte](./oop/classes.ts)
  - Beispiel: Employee-Klasse
- [Vererbung: Klassen "erben" Eigenschaften und Methoden von Elternklassen](./oop/inheritance.ts)
  - Beispiel: Manager-Klasse erbt Eigenschaften von Employee-Klasse und fügt neue Eigenschaften hinzu
- [Polymorphie: Objekte verschiedener Klassen verfügen über die gleiche "Schnittstelle", machen aber unterschiedliche Dinge](./oop/polymorphism.ts)
  - Beispiel: Manager-Klasse "überschreibt" die Methode `calculateSalary`: Wenn ein Employee ein Manager ist, wird das Gehalt anders berechnet.
