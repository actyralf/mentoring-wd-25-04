## EJS Template Engine with Express

### What is EJS?

EJS (Embedded JavaScript) is a simple templating engine for Node.js that lets you generate HTML using plain JavaScript inside templates.

It allows you to:

- Embed JavaScript directly into HTML
- Render dynamic data on the server
- Separate presentation (views) from logic (routes/controllers)

### Main Features

- ✅ Embed JavaScript inside HTML (<%= %>, <% %>)
- ✅ Pass data from Express to templates
- ✅ Reusable partials (header, footer, etc.)
- ✅ Works naturally with Express

### Setup with Express

1️⃣ Install

```zsh
npm install ejs
```

2️⃣ Configure Express

```js
import express from "express";

const app = express();

app.set("view engine", "ejs");
```

By default:

- Templates are placed inside a `/views` folder
- Files use the .ejs extension

Example structure:

```code
project/
  views/
    index.ejs
```

### Basic Usage

1️⃣ Route

```js
app.get("/", (req, res) => {
  res.render("index", { name: "Ralf" });
});
```

2️⃣ Template (views/index.ejs)

```html
<!DOCTYPE html>
<html>
  <body>
    <h1>Hello <%= name %></h1>
  </body>
</html>
```

<%= %> outputs escaped content.

3️⃣ Example loop

```html
<ul>
  <% movies.forEach(movie => { %>
  <li><%= movie %></li>
  <% }) %>
</ul>
```

### Summary

EJS is a lightweight, server-side templating engine for Express that allows you to embed JavaScript in HTML to render dynamic content. It’s simple to set up, easy to learn, and ideal for traditional server-rendered applications.
